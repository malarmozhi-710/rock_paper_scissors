# rock_paper_scissors
To design and implement a simple Rock Paper Scissors game where the user plays against the computer. The program should take user input, generate a random choice for the computer, and display the result (win, lose, or draw). 

This project is a simple Rock Paper Scissors game. The user selects one option among Rock, Paper, or Scissors. The computer randomly selects its choice.
The program then compares both choices and decides the result:
•	If both choices are the same → Draw 
•	If the user’s choice is stronger → User wins 
•	Otherwise → Computer wins 
This game demonstrates basic programming concepts like input handling, conditional logic, and random number generation
